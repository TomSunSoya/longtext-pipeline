# Code of Conduct

## Our standard

This project is committed to providing a welcoming, professional, and harassment-free experience for everyone participating in the community.

Examples of behavior that contribute to a positive environment include:

- Being respectful in disagreements
- Assuming good intent while still being precise and direct
- Giving actionable technical feedback
- Sharing context when reporting bugs or requesting changes

Examples of unacceptable behavior include:

- Harassment, discrimination, or intimidation
- Personal attacks
- Deliberate trolling or repeated bad-faith engagement
- Publishing private information without permission

## Scope

This Code of Conduct applies to GitHub issues, pull requests, discussions, release notes, and any other official project spaces.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer privately via GitHub: <https://github.com/TomSunSoya>.

## Enforcement

The maintainer may remove comments, close discussions, reject contributions, or block participants whose behavior violates this policy.
